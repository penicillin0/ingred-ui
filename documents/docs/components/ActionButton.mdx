# ActionButton
データの登録や編集、削除といったアクションを含む動作をする時に用いるボタンです。  
テーブルセルの中など比較的コンパクトな場所に設置することができます。

## 種類

### サイズ

### 原則

#### Good

:::tip

Use the power of React to create interactive blog posts.

```js
<ActionButton icon="pencil" color="primary">
  Sample
</ActionButton>
```

<ThemeProvider>
  <Flex display="flex">
    <Spacer mr={5}>
      <Typography weight="bold">Primary</Typography>
      <Spacer pt={1} />
      <ActionButton icon="pencil" color="primary">
        Edit
      </ActionButton>
    </Spacer>
    <Spacer mr={5}>
      <Typography weight="bold">Warning</Typography>
      <Spacer pt={1} />
      <ActionButton icon="pencil" color="warning">
        Edit
      </ActionButton>
    </Spacer>
    <Spacer mr={5}>
      <Typography weight="bold">Disabled</Typography>
      <Spacer pt={1} />
      <ActionButton icon="pencil" disabled={true}>
        Edit
      </ActionButton>
    </Spacer>
  </Flex>
</ThemeProvider>
:::

```js
<ActionButton icon="pencil" color="secondary">
```
